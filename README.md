<pre>
<h3>Description</h3>
<code>
  It converts user input datetime string into user defined datetime string with formatting done on user input
</code>
</pre>

<pre>
<h3>Example</h3>
<code>
  <b>input</b>: ('2019-18-03 10:03:11', '%y-%d-%m h:s:m', '%y/%m/%d h:m:s')
  <b>output</b>: ('2019/03/18 10:11:03')
</code>
</pre>
